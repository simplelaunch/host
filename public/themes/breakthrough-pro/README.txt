BREAKTHROUGH PRO
https://my.studiopress.com/themes/breakthrough/

INSTALL
1. Upload the Breakthrough Pro theme folder via FTP to your wp-content/themes/ directory. (The Genesis parent theme needs to be in the wp-content/themes/ directory as well.)
2. Go to your WordPress dashboard and select Appearance.
3. Activate the Breakthrough Pro theme.
4. Inside your WordPress dashboard, go to Genesis > Theme Settings and configure them to your liking. To set up the theme like the demo, please visit https://my.studiopress.com/documentation/breakthrough-pro-theme/.
